## Created with
- Next.js
- TailwindCSS
- API: mockapi.io [https://5eed24da4cbc340016330f0d.mockapi.io/api]

## Getting Started

First, install dependencies:
```bash
npm install
# or
yarn
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
